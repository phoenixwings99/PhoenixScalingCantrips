# Phoenix's Scaling Cantrips
This mod is a replacement for Phoenix's Cantrips and More Cantrips. 

## Features
Melee touch cantrips for all five elemental damage types.
Ranged touch cantrip for sonic.
Scaling (1 die every two levels, max six) for all damage cantrips.
Scaling (1 temp hp per level, max 10) for virtue.
Proliferation of elemental cantrips to Oracle/Witch/Shaman mysteries/curses/patrons/spirits as appropriate.
Fixing Magus ranged touch access.
Arrowsong Minstrel (Character Options Plus) gets sonic attack cantrips.
